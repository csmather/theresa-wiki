# theresa-wiki

Theresa's personal knowledge wiki — grad school (MSW) and general learning, distilled into linked topic pages that compound over time.

Built with [Quartz 5](https://quartz.jzhao.xyz/), published at **https://csmather.github.io/theresa-wiki** via GitHub Pages on every push to `main`.

- `content/maps/` — living topic pages
- `content/sources/` — one summary page per ingested source
- Ingestion is curated by hand (well, by Claude) — see `CLAUDE.md`
